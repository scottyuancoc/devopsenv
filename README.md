# DevOpsEnv

DevOpsEnv is an DevOps environment confirgurator for getting an DevOps development ready for a developer to do generic 
development work using following techologies:

* GIT / GIT-Bash
* VS Code 
* Java SE 11/Maven
* NodeJS/Npm
